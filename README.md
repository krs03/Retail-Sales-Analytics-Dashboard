# 🛒 Retail Sales Analytics Dashboard

## 📌 Project Overview
The Retail Sales Analytics Dashboard project aims to analyze retail sales data and generate actionable business insights using Data Analytics, SQL, Python, and Power BI.

This project is being developed as part of the Infotact Technical Internship Program.

# 🎯 Objectives
- Analyze retail sales performance
- Identify top-performing products and regions
- Track sales trends over time
- Monitor profit and revenue metrics
- Build an interactive Power BI dashboard
- Generate business insights for decision-making

# 🛠 Technologies Used

| Technology | Purpose |
| Python | Data Cleaning & Analysis |
| Pandas | Data Manipulation |
| SQL | Querying & KPI Analysis |
| Power BI | Dashboard Visualization |
